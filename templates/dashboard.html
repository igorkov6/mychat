Игорь Ковалев FPW-84

Ваше задание очень простое:

реализовать базовый мессенджер со следующими функциями:

  • отправка и получение сообщений;
  • создание, редактирование и удаление групповых чатов и переписка в них (подсказка — управлять чатами лучше по REST API, а переписываться так же, как в обычных чатах, но с использованием на сервере идеологии «комнат»);
  • редактирование личной информации пользователя (имя и аватар);
  • просмотр списка других пользователей с переходом на отправку им сообщений.

Для реализации проекта использованы следующие технологии:

django channels
передача сообщений через Winsocket
rest framework
управление групповыми чатами
social auth
авторизация через соцсети
PostgreSQL psycopg2
основная база данных
python dotenv
хранение конфиденциальной информации
python bleach
фильтрация входящих сообщений
redis
хранение промежуточных данных
pillow
работа с изображениями
messages
всплывающие сообщения
